# Práctica de k-Nearest Neighbors (KNN)

Esta práctica consiste en la implementación y experimentación con el algoritmo de k-Nearest Neighbors.

## Contenido

*   `knn_practica.ipynb`: Jupyter Notebook con la implementación del algoritmo de KNN.
*   `emails.csv`: Dataset de correos electrónicos para clasificación.
*   `iris.csv`: Dataset de flores de iris para clasificación.
